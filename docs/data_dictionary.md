# Data Dictionary

## Raw sampled events (`data/raw/events_*.csv`)

Generated by `src/utils/sample_events.py` from the [Kaggle e-commerce behavior dataset](https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store).

| Column         | Type    | Description |
|----------------|---------|-------------|
| `event_id`     | string  | Unique ID added during sampling (`evt_{seed}_{index}`) |
| `event_time`   | string  | Event timestamp (UTC) |
| `event_type`   | string  | `view`, `cart`, `remove_from_cart`, or `purchase` |
| `product_id`   | integer | Product identifier |
| `category_id`  | integer | Category identifier |
| `category_code`| string  | Category taxonomy code (nullable) |
| `brand`        | string  | Brand name, lowercase (nullable) |
| `price`        | float   | Product price |
| `user_id`      | integer | User identifier |
| `user_session` | string  | Session identifier |

### Sample files

| File | Rows | Use case |
|------|------|----------|
| `data/raw/events_1m.csv` | 1,000,000 | Default local + cloud-lite demo |
| `data/raw/events_5m.csv` | 5,000,000 | Extended local stress test (**no cloud reload**) |

Source monthly files live in `data/raw/source/` and are **not** uploaded to S3 or Snowflake.

## Planned layers

| Layer  | Location              | Description                          |
|--------|-----------------------|--------------------------------------|
| Raw    | `data/raw/`           | Sampled CSV event files (1M / 5M)    |
| Bronze | `data/bronze/`        | Kafka-consumed events as Parquet     |
| Silver | `data/silver/`        | Cleaned, deduplicated events         |
| Gold   | `data/gold/`          | Session, funnel, and revenue marts   |
